# design_system_widgetbook

Dev-only interactive component catalog for `packages/design_system` — never
shipped to production, not referenced by any `melos run build:*` script.
Complements the real `matchesGoldenFile` golden tests in
`packages/design_system/test/golden/` rather than replacing them: golden
tests are automated, CI-enforced pixel-diff regression detection; this is a
manual browsing/review tool for humans (light/dark toggle via the Theme
addon, one use case per widget state).

## Run it

```
cd apps/widgetbook
fvm flutter run -d chrome
```

## Add a new use case

Add a `@widgetbook.UseCase(name: '...', type: SomeWidget)`-annotated
top-level function to `lib/use_cases/`, then run `melos run gen` — the
directory tree (`lib/main.directories.g.dart`) is generated by
`widgetbook_generator` from every annotated function it finds.
